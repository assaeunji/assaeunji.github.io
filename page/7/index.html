Metropolis-Hastings Algorithm

Posted on 2019-06-30 Edited on 2023-05-02 In Bayesian
  • 메트로폴리스-헤이스팅스 알고리즘 (Metropolis-Hasting Algorithm)은 사후분포를 analytic하게 구하기 어려울 때 사용하는 방법이다. 이 알고리즘의 기본 아이디어는 마코프 체인의 정상분포가 사후분포인 f(x)f(x)f(x)가 되도록 마코프 체인을 생성하는 것이다. 그렇다면 마코프 체인과 정상분포란 무엇일까?

Markov Chain

  • 마코프 성질 (Markov Property): 마코프 체인 {X(t)}\{X^{(t)}\}{X(t)}는 독립이 아닌 확률 표본의 sequence이며, 바로 직전의 값에만 의존한다.

X(0),X(1),X(2),…,X(t),…such thatX(t+1)∣X(t),…,X(0)=DX(t+1)∣X(t)\begin{aligned} X^{(0)}, X^{(1)}, X^{(2)},\ldots, X{(t)},\ldots\\ \text{such\ that} X^{(t+1)}|X^{(t)},\ldots,X^{(0)} \overset{\mathcal{D}}{=} X^{(t+1)}|X^{(t)} \end{aligned} X(0),X(1),X(2),…,X(t),…such thatX(t+1)∣X(t),…,X(0)=DX(t+1)∣X(t)​

마코프 커널 (Markov Kernel): $$X^{(t+1)}\vert X^{(t)}$$의 조건부확률

X(t+1)∣X(t),…,X(0)∼K(X(t),X(t+1))X^{(t+1)}|X^{(t)},\ldots, X^{(0)} \sim K(X^{(t)},X^{(t+1)}) X(t+1)∣X(t),…,X(0)∼K(X(t),X(t+1))

  • 예시: X(t+1)=X(t)+etX^{(t+1)} = X^{(t)} + e_tX(t+1)=X(t)+et​, et∼N(0,1)e_t \sim N(0,1)et​∼N(0,1)을 만족하는 임의 보행 마코프체인을 고려해보자. 이때 마코프 커널은 K(X(t),X(t+1))=N(X(t),1)K(X^{(t)},X^{(t+1)}) = N(X^{(t)},1)K(X(t),X(t+1))=N(X(t),1)이다.

마코프 체인은 세가지 성질을 만족해야 한다.

  1. Irreducible: 어떤 초기값 X(0)X^{(0)}X(0)을 갖던 간에 다른 상태로 가는 확률이 항상 양수여야 한다. 즉, 모든 상태들이 연결이 되어야 한다.
  2. Aperiodic: 모든 상태들이 주기를 가지고 있지 않아야 한다.
  3. Recurrent: 수열이 어떤 상태이든 무한하게 도달할 수 있다. 즉, 특정한 상태에 재방문이 가능해야 한다.

이러한 세가지 성질을 만족하면 마코프 체인이 ergodic하다 정의한다. ergodic한 마코프체인은 정상 분포를
지닌다. 그렇다면 정상성은 무엇일까?

Stationarity

  • 정상성(Stationarity)을 지닌 분포는 다음의 성질을 만족한다:

If X(t)∼f⇒X(t+1)∼f\text{If}\ X^{(t)}\sim f \Rightarrow X^{(t+1)} \sim f If X(t)∼f⇒X(t+1)∼f

  • 즉, {X(t)}\{X^{(t)}\}{X(t)}의 분포가 어떤 확률분포 f(x)f(x)f(x)로 수렴하는 성질을 일컫는다.

Ergodic Theorem

  • 개인적으로 MCMC에서 나오는 영단어를 이해하기가 어렵다고 생각한다. 앞서 말했듯이 "ergodic"하다는 것은 마코프 체인의 성질이고, 정상분포가 되기 위한 조건이다.
  • 커널 KKK가 정상분포 fff를 가진 ergodic한 마코프 체인 {X(t),t=1,…,T}\{X^{(t)}, t=1,\ldots, T\}{X(t),t=1,…,T}을 생성했다 가정하자. 이 때 마코프 체인 {X(t),t=1,…,T}\{X^{(t)}, t=1,\ldots, T\}{X(t),t=1,…,T}는 fff에서 난수를 생성한 것과 같다는 것이 Ergodic theorem의 핵심이다.
  • 이를 수식으로 표현하면 SLLN (Strong Law of Large Number)으로 불린다.

1T∑t=1Th(X(t))→Ef(h(X))\frac{1}{T}\sum_{t=1}^T h(X^{(t)}) \to \mathbb{E}_f(h(X)) T1​t=1∑T​h(X(t))→Ef​(h(X))

Metropolis-Hastings Algorithm

  • 이제 정상성을 지닌 마코프체인을 생성하는 방법인 MH 알고리즘에 대해 알아보자.

  • MH 알고리즘은 다음과 같은 거절법으로 마코프 체인 {X0,X1,…,}\{X_0, X_1,\ldots,\}{X0​,X1​,…,}을 생성한다.

    1. 제안 분포 (proposal density) g(⋅∣X(t))g(\cdot\vert X^{(t)})g(⋅∣X(t)) 선택: 보통 제안 분포는 목표 분포 (target density)와 범위가 같은 것을 선택한다.
    2. g(⋅∣X(t))g(\cdot\vert X^{(t)})g(⋅∣X(t))로부터 YYY 생성
    3. Unif(0,1)\text{Unif}(0,1)Unif(0,1)로부터 UUU 생성
    4. 만약 U≤α(X(t),Y)U \le \alpha(X^{(t)},Y)U≤α(X(t),Y)이면 YYY를 채택해 X(t+1)=YX^{(t+1)} = YX(t+1)=Y로, 그렇지 않으면 X(t+1)=X(t)X^{(t+1)} = X^{(t)}X(t+1)=X(t)로 설정.

    마코프 체인이 정상분포로 수렴할 때까지 2 ~ 4를 반복한다.

  • 여기서 채택 확률인 α(X(t),Y)\alpha(X^{(t)},Y)α(X(t),Y)는 다음과 같이 정의된다:

α(X(t),Y)=min(f(Y)∣g(X(t)∣Y)f(X(t))g(Y∣X(t)))\alpha(X^{(t)},Y)=\text{min}\left(\frac{f(Y)|g(X^{(t)}|Y)}{f(X^{(t)}) g(Y|X^{(t)})}\right) α(X(t),Y)=min(f(X(t))g(Y∣X(t))f(Y)∣g(X(t)∣Y)​)

  • 이를 통한 마코프 체인은 정상분포 f(x)f(x)f(x)를 갖는다.
  • MH 알고리즘은 독립표집기와 무작위 표집기를 주로 이용한다.

독립 표집기 (Independent sampler)

  • g(Y∣X(t))=g(Y)g(Y \vert X^{(t)}) = g(Y)g(Y∣X(t))=g(Y)
  • 제안분포 g(⋅∣X(t))g(\cdot \vert X^{(t)})g(⋅∣X(t))가 X(t)X^{(t)}X(t)에 의존하지 않는 형태
    4* Y∼g(y)Y \sim g(y)Y∼g(y) 생성
  • 다음의 기준으로 YYY 채택:

X(t+1)={Yt,with probability α(X(t),Y)X(t),otherwise.X^{(t+1)} = \begin{cases} Y_t, &\text{with\ probability}\ \alpha(X^{(t)},Y)\\ X^{(t)}, & \text{otherwise}. \end{cases} X(t+1)={Yt​,X(t),​with probability α(X(t),Y)otherwise.​

  • 채택확률은 다음과 같이 정의된다.

α(X(t),Y)=min(1,f(Y)/g(Y)f(X(t))/g(X(t)))=min(1,f(Y)g(X(t))f(X(t))g(Y))\alpha(X^{(t)},Y) = \text{min}\left(1,\frac{f(Y)/g(Y)}{f(X^{(t)})/g(X^{(t)})} \right) = \text{min}\left(1,\frac{f(Y)g(X^{(t)})}{f(X^{(t)})g(Y)} \right) α(X(t),Y)=min(1,f(X(t))/g(X(t))f(Y)/g(Y)​)=min(1,f(X(t))g(Y)f(Y)g(X(t))​)

무작위보행 표집기(Random Walk sampler)

  • 제안분포로 g(Y∣X(t))=g(X(t)∣Y)g(Y\vert X^{(t)}) = g(X^{(t)} \vert Y)g(Y∣X(t))=g(X(t)∣Y)를 만족하는 대칭 분포 사용
  • 대표적으로 Y∣X(t)∼(X(t),b2), b>0Y\vert X^{(t)} \sim (X^{(t)},b^2),\ b>0Y∣X(t)∼(X(t),b2), b>0을 이용할 수 있다:

Y=X(t)+ε,ε∼N(0,b2)Y=X^{(t)}+\varepsilon, \varepsilon \sim \mathcal{N}(0, b^2) Y=X(t)+ε,ε∼N(0,b2)

  • 이 때의 채택확률은 다음과 같다:

α(X(t),Y)=min(1,f(Y)f(X(t))⋅g(X(t)∣Y)g(Y∣X(t)))=min(1,f(Y)f(X(t)))\alpha(X^{(t)},Y) = \text{min}\left(1, \frac{f(Y)}{f(X^{(t)})}\cdot \frac{g(X^{(t)}\vert Y)}{g(Y\vert X^{(t)})}\right) = \text{min} \left(1,\frac{f(Y)}{f(X^{(t)})}\right) α(X(t),Y)=min(1,f(X(t))f(Y)​⋅g(Y∣X(t))g(X(t)∣Y)​)=min(1,f(X(t))f(Y)​)
